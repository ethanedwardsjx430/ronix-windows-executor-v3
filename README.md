## About Ronix

Ronix is a Windows-focused Roblox script executor built for users who want a compact desktop utility for running Lua scripts and handling a script hub from a single interface. The emphasis is on a clean UI, quick access to execution controls, and a tool that keeps resource usage low.

It brings together one-click execution, batch script runs, and a persistent queue stored in SQLite so repeated actions are easier to manage. With auto-update support, built-in debugging tools, and a multi-language desktop interface, Ronix is aimed at practical Roblox scripting workflows on current systems.
